# Fed PR Lender Portal
This portal features a structured 4-step workflow for OTP verification, agent assessment, customer KYC verification, and gold jewel details.

## 🚀 Live Demo
You can view the live interactive prototype here:
**[View Live Website](https://ritiksahoo.github.io/Fed-PR-Lender-Portal/)**

## Features
- **4-Step Sequential Workflow**: OTP Verification → Agent Details → Customer KYC → Gold Jewels.
- **High-Fidelity UI**: Designed with Federal Bank and Rupeek branding.
- **Real-Time Timers**: Functional OTP resend timers and individual verification segments.
- **Interactive Metrics**: Dynamic card-based layouts for Current Loan Details.
- **Digital Appraisal**: Click-to-view and downloadable appraisal sheets.
- **Optimized for Scanning**: High-density tables and clear hierarchical labels.

## Project Structure
- `index.html`: The core structure and 4-step container logic.
- `style.css`: Premium CSS styling with custom theme variables.
- `script.js`: Interactive logic for navigation, timers, and transitions.
- `/assets`: High-resolution placeholders for logos, profiles, and appraisal sheets.
